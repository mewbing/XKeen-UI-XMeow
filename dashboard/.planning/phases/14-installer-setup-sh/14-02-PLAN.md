  <action>
**Step 1: Create `.gitattributes`** in the project root with:
```
setup.sh text eol=lf
```
This ensures setup.sh always has LF line endings in git, even on Windows. CRLF on the router causes "set: line 5: illegal option -" error in BusyBox ash.

**Step 2: Create `setup.sh`** -- a POSIX sh compatible install-only script. The entire script MUST be wrapped in `main() { ... }; main "$@"` for curl|sh pipe safety. MUST use ONLY POSIX sh constructs (NO bashisms: no `[[ ]]`, no arrays, no `source`, no process substitution, no `function` keyword).

**Script structure and behavior:**

1. **Shebang and safety:** `#!/bin/sh` then `set -eu` inside main(). Note: do NOT put `set -e` before main() -- it must be inside the wrapper function.

2. **Constants:**
   ```
   REPO="mewbing/XKeen-UI-Xmeow"
   INSTALL_DIR="/opt/etc/xmeow-ui"
   BIN_NAME="xmeow-server"
   CONF_FILE="$INSTALL_DIR/xmeow-ui.conf"
   INITD_SCRIPT="/opt/etc/init.d/S99xmeow-ui"
   PIDFILE="/opt/var/run/xmeow-ui.pid"
   MIHOMO_DIR="/opt/etc/mihomo"
   EXTUI_DIR="$MIHOMO_DIR/ui"
   TMP_DIR="/tmp/xmeow-install"
   ```

3. **ANSI color helpers** (POSIX printf):
   ```
   RED='\033[0;31m'  GREEN='\033[0;32m'  YELLOW='\033[1;33m'
   BLUE='\033[0;34m'  CYAN='\033[0;36m'  BOLD='\033[1m'  NC='\033[0m'
   ```
   Functions: `info()`, `success()`, `warn()`, `error_msg()`, `die()` -- use printf with color codes. Name the error function `error_msg` to avoid collision with shell builtins.

4. **Language detection:**
   ```
   detect_lang() -- check $LANG for ru_* prefix, set LANG_CODE=ru or en
   msg() -- takes two args ($1=ru, $2=en), prints based on LANG_CODE
   ```

5. **ASCII logo** -- `print_logo()` function, simple XMeow branding (3-5 lines of ASCII art), printed with printf and CYAN color. Keep it minimal.

6. **Root check:** Verify running as root (`[ "$(id -u)" -ne 0 ]` then die with error).

7. **Architecture detection** -- `detect_arch()`:
   - `uname -m` -> case statement:
     - `aarch64` -> `ARCH=arm64`
     - `armv7l` -> `ARCH=armv7`
     - `x86_64` -> `ARCH=amd64`
     - `mips*` -> Two-stage MIPS detection:
       - Stage 1: `grep -qi "mipsel" /opt/etc/opkg.conf 2>/dev/null` -> `ARCH=mipsle`
       - Stage 1 alt: else if `grep -qi "mips" /opt/etc/opkg.conf` -> `ARCH=mips`
       - Stage 2 fallback: `echo -n I | od -to2 | head -n1 | cut -f2 -d" " | cut -c6` -- if "1" then mipsle, else mips
     - `*` -> die "Unsupported architecture"
   - Set `SUFFIX="linux_${ARCH}"`
   - Print detected architecture with info()

8. **Get latest release** -- `get_latest()`:
   - Fetch `https://api.github.com/repos/$REPO/releases/latest` with `curl -fsSL`
   - Extract tag_name using grep/sed: `grep '"tag_name"' | head -1 | sed 's/.*"tag_name" *: *"\([^"]*\)".*/\1/'`
   - Strip leading 'v' for version number
   - Construct download URLs:
     - `BINARY_URL=https://github.com/$REPO/releases/download/$TAG/xmeow-server_${VERSION}_${SUFFIX}.tar.gz`
     - `CHECKSUM_URL=https://github.com/$REPO/releases/download/$TAG/checksums.txt`
     - `DISTUI_URL=https://github.com/$REPO/releases/download/$TAG/dist.tar.gz`
   - Print version found with info()

9. **Download and verify** -- `download_and_verify()`:
   - Create temp dir: `mkdir -p "$TMP_DIR"`
   - Download binary archive with `curl --progress-bar --retry 3 -fSL -o "$TMP_DIR/$ARCHIVE_NAME" "$BINARY_URL"`
   - Download checksums.txt with `curl -fsSL -o "$TMP_DIR/checksums.txt" "$CHECKSUM_URL"`
   - Verify SHA256: extract expected hash for our archive from checksums.txt with grep+awk, compute actual with `sha256sum`, compare. Die on mismatch.
   - Extract binary from tar.gz: `tar -xzf "$TMP_DIR/$ARCHIVE_NAME" -C "$TMP_DIR" "$BIN_NAME"`
   - Print checksum OK with success()

10. **Install binary** -- `install_binary()`:
    - `mkdir -p "$INSTALL_DIR"`
    - Copy binary: `cp "$TMP_DIR/$BIN_NAME" "$INSTALL_DIR/$BIN_NAME"`
    - Make executable: `chmod +x "$INSTALL_DIR/$BIN_NAME"`
    - Verify it runs: `"$INSTALL_DIR/$BIN_NAME" --version` and print the output
    - Print success with success()

11. **Install external-UI** -- `install_extui()`:
    - Download dist.tar.gz with `curl --progress-bar --retry 3 -fSL -o "$TMP_DIR/dist.tar.gz" "$DISTUI_URL"`
    - Verify checksum (same checksums.txt already downloaded)
    - `mkdir -p "$EXTUI_DIR"`
    - Extract: `tar -xzf "$TMP_DIR/dist.tar.gz" -C "$EXTUI_DIR"`
    - Update mihomo config.yaml to set `external-ui: ui` (relative path, NOT absolute -- mihomo restricts paths to workdir):
      - If `grep -q "^external-ui:" "$MIHOMO_CONFIG"` then `sed -i "s|^external-ui:.*|external-ui: ui|" "$MIHOMO_CONFIG"`
      - Else append `echo "external-ui: ui" >> "$MIHOMO_CONFIG"`
    - Warn if `external-ui-url` is set: if `grep -q "^external-ui-url:" "$MIHOMO_CONFIG"` then warn that it will overwrite local files on restart and suggest commenting it out.
    - Print success with success()

12. **Create config file** -- `create_config()`:
    - Only create if NOT already exists (preserve user settings on reinstall)
    - Write to `$CONF_FILE`:
      ```
      PORT=5000
      MIHOMO_CONFIG_PATH=/opt/etc/mihomo/config.yaml
      XKEEN_DIR=/opt/etc/xkeen
      XKEEN_LOG_DIR=/opt/var/log/xray
      ```
    - Print success/skip message

13. **Create init.d script** -- `create_initd()`:
    - Always overwrite (structure may change between versions, per CONTEXT.md decision)
    - Write `/opt/etc/init.d/S99xmeow-ui` with cat heredoc:
      ```sh
      #!/bin/sh
      CONF="/opt/etc/xmeow-ui/xmeow-ui.conf"
      BIN="/opt/etc/xmeow-ui/xmeow-server"
      PIDFILE="/opt/var/run/xmeow-ui.pid"

      [ -f "$CONF" ] && . "$CONF"

      export PORT="${PORT:-5000}"
      export MIHOMO_CONFIG_PATH="${MIHOMO_CONFIG_PATH:-/opt/etc/mihomo/config.yaml}"
      export XKEEN_DIR="${XKEEN_DIR:-/opt/etc/xkeen}"
      export XKEEN_LOG_DIR="${XKEEN_LOG_DIR:-/opt/var/log/xray}"

      case "$1" in
          start)
              if [ -f "$PIDFILE" ] && kill -0 "$(cat "$PIDFILE")" 2>/dev/null; then
                  echo "xmeow-ui already running (PID $(cat "$PIDFILE"))"
              else
                  echo "Starting xmeow-ui..."
                  "$BIN" > /dev/null 2>&1 &
                  echo $! > "$PIDFILE"
                  echo "xmeow-ui started (PID $!)"
              fi
              ;;
          stop)
              if [ -f "$PIDFILE" ]; then
                  PID=$(cat "$PIDFILE")
                  if kill -0 "$PID" 2>/dev/null; then
                      echo "Stopping xmeow-ui (PID $PID)..."
                      kill "$PID"
                      rm -f "$PIDFILE"
                  else
                      echo "xmeow-ui not running (stale PID file)"
                      rm -f "$PIDFILE"
                  fi
              else
                  echo "xmeow-ui not running"
              fi
              ;;
          restart)
              "$0" stop
              sleep 1
              "$0" start
              ;;
          status)
              if [ -f "$PIDFILE" ] && kill -0 "$(cat "$PIDFILE")" 2>/dev/null; then
                  echo "xmeow-ui running (PID $(cat "$PIDFILE"))"
              else
                  echo "xmeow-ui not running"
              fi
              ;;
          *)
              echo "Usage: $0 {start|stop|restart|status}"
              exit 1
              ;;
      esac
      ```
    - `chmod +x "$INITD_SCRIPT"`
    - Make sure PID directory exists: `mkdir -p /opt/var/run`
    - Print success

14. **Start and verify** -- `start_and_verify()`:
    - Stop existing instance if running: `"$INITD_SCRIPT" stop 2>/dev/null || true`
    - Start service: `"$INITD_SCRIPT" start`
    - Sleep 2 seconds for startup
    - Verify process is running: check PID file and `kill -0`
    - Attempt HTTP health check: `curl -sf "http://127.0.0.1:${PORT:-5000}/api/health" > /dev/null 2>&1` -- if fails, warn but don't die (mihomo may not be configured yet)
    - Print success

15. **Print success message** -- `print_success()`:
    - Auto-detect IP via `ip route get 1 2>/dev/null | grep -o 'src [0-9.]*' | awk '{print $2}'` with fallback to `hostname -I` then `<router-ip>`
    - Print formatted message:
      ```
      ====================================
       XMeow UI v{VERSION} installed!
      ====================================
       Dashboard: http://{IP}:5000
       Service:   /opt/etc/init.d/S99xmeow-ui {start|stop|restart|status}
       Config:    /opt/etc/xmeow-ui/xmeow-ui.conf
      ====================================
      ```
    - Use GREEN color for the success block

16. **Cleanup** -- `cleanup()`:
    - `rm -rf "$TMP_DIR"`

17. **main() flow:**
    ```
    main() {
        set -eu
        detect_lang
        print_logo
        [ "$(id -u)" -ne 0 ] && die "Root required / ..."

        msg "Установка XMeow UI..." "Installing XMeow UI..."
        detect_arch
        get_latest
        download_and_verify
        install_binary
        install_extui
        create_config
        create_initd
        start_and_verify
        cleanup
        print_success
    }
    main "$@"
    ```

**CRITICAL CONSTRAINTS:**
- NO interactive menu (no read, no prompt, no choices) -- this is install-only
- NO update flow, NO uninstall flow (those are deferred to dashboard UI)
- ALL sh constructs must be POSIX compatible (BusyBox ash)
- The script MUST have LF line endings (verified by .gitattributes)
- Use `printf` not `echo -e` for escape sequences (echo -e is a bashism on some systems)
- Use `[ ]` not `[[ ]]` for tests
- Use `. file` not `source file`
- Use `command -v` not `which`
- external-ui path in mihomo config must be relative (`ui`) not absolute (mihomo security restriction)
