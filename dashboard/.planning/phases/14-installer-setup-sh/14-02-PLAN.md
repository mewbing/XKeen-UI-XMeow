<task type="auto">
  <name>Task 1: Create setup.sh installer script</name>
  <files>setup.sh</files>
  <action>
Create `setup.sh` in project root as a POSIX sh script. MUST use wrapper function pattern (`main() { ... }; main "$@"`) for curl|sh safety. NO bashisms (no `[[ ]]`, no arrays, no `source`, no process substitution).

**Script structure** (all functions inside main wrapper):

1. **Constants and Config:**
   ```
   REPO="mewbing/XKeen-UI-Xmeow"
   API_URL="https://api.github.com/repos/$REPO/releases/latest"
   INSTALL_DIR="/opt/etc/xmeow-ui"
   BIN_NAME="xmeow-server"
   BIN_PATH="$INSTALL_DIR/$BIN_NAME"
   CONF_PATH="$INSTALL_DIR/xmeow-ui.conf"
   INITD_PATH="/opt/etc/init.d/S99xmeow-ui"
   PIDFILE="/opt/var/run/xmeow-ui.pid"
   MIHOMO_DIR="/opt/etc/mihomo"
   EXTUI_DIR="$MIHOMO_DIR/ui"
   TMP_DIR="/tmp/xmeow-install"
   ```

2. **Color helpers:** `info()`, `success()`, `warn()`, `error()`, `die()` — ANSI colors via printf, NOT echo.

3. **Language detection:** `detect_lang()` — check `$LANG` for `ru_*` pattern. `msg()` function taking two args (ru, en), prints based on $LANG_CODE.

4. **ASCII logo:** `print_logo()` — simple XMeow branding text with version. Use cyan/bold ANSI.

5. **TTY detection:** `detect_tty()` — set `READ_CMD` strategy. If `[ -t 0 ]` is true, read from stdin. Otherwise read from `/dev/tty`. Create `prompt()` function that handles both cases.

6. **Architecture detection:** `detect_arch()` — use two-stage MIPS detection:
   - `uname -m` mapping: aarch64->arm64, armv7l->armv7, x86_64->amd64
   - For `mips*`: Stage 1 — parse `/opt/etc/opkg.conf` with `grep -qi "mipsel"`. Stage 2 — `echo -n I | od -to2` endianness test.
   - Set global `ARCH` variable.
   - die on unsupported architecture.

7. **HTTP client:** `download()` — check for curl first, fallback to wget. For progress downloads use `curl --progress-bar -fSL` or `wget -q --show-progress`. For silent API calls use `curl -fsSL` or `wget -qO-`.

8. **GitHub API:** `get_latest()` — fetch latest release JSON via API, parse `tag_name` with grep/sed (no jq). Set `LATEST_VER`, `LATEST_VER_NUM` (strip leading v), construct `DOWNLOAD_URL`, `CHECKSUM_URL`, `DISTUI_URL`.

9. **Checksum verification:** `verify_checksum()` — download checksums.txt, extract expected hash for filename, compare with `sha256sum`. Die on mismatch.

10. **IP detection:** `get_local_ip()` — try `ip route get 1`, fallback to `hostname -I`, fallback to `ifconfig`. Return first non-127 IP.

11. **Config file creation:** `create_config()` — write xmeow-ui.conf ONLY if it doesn't exist (preserve user changes). Contents:
    ```
    PORT=5000
    MIHOMO_CONFIG_PATH=/opt/etc/mihomo/config.yaml
    XKEEN_DIR=/opt/etc/xkeen
    XKEEN_LOG_DIR=/opt/var/log/xray
    ```

12. **init.d script creation:** `create_initd()` — ALWAYS overwrite (even on update). Write S99xmeow-ui with:
    - Source .conf file
    - Export env vars (PORT, MIHOMO_CONFIG_PATH, XKEEN_DIR, XKEEN_LOG_DIR)
    - case statement: start (background + PID file), stop (kill by PID + pidof fallback), restart, status
    - PID liveness check with `kill -0` before trusting PID file
    - chmod +x

13. **External-UI installation:** `install_extui()` — mkdir -p extui dir, download dist.tar.gz with progress, verify checksum, extract to EXTUI_DIR. Then update mihomo config.yaml: if `external-ui:` line exists, sed replace; if not, append. Warn if `external-ui-url:` is detected (suggest commenting out).

14. **do_install():**
    - Check if already installed (warn, suggest update)
    - detect_arch
    - get_latest
    - mkdir -p TMP_DIR, INSTALL_DIR
    - Download binary tar.gz with progress to TMP_DIR
    - verify_checksum
    - Extract binary from tar.gz, move to BIN_PATH, chmod +x
    - create_config
    - create_initd
    - install_extui
    - Start service via init.d
    - Wait 2 seconds, verify service running (check PID or curl health endpoint)
    - print_success with IP, port, version, commands
    - Clean up TMP_DIR

15. **do_update():**
    - Check if installed (die if not)
    - Get current version: `$BIN_PATH --version`
    - get_latest
    - Compare versions (simple string != comparison)
    - If same version, inform and exit
    - Download new binary to TMP_DIR
    - verify_checksum
    - Stop service
    - Backup: cp BIN_PATH to BIN_PATH.OLD_VERSION
    - Extract and replace binary
    - create_initd (overwrite)
    - Start service
    - Verify running
    - print_success
    - Clean up

16. **do_uninstall():**
    - Check if installed (die if not)
    - Stop service
    - Remove BIN_PATH
    - Remove INITD_PATH
    - Ask user (via prompt()) whether to remove config and backups
    - If yes: rm -rf INSTALL_DIR
    - If no: keep INSTALL_DIR with config
    - Do NOT touch mihomo files or EXTUI_DIR
    - Print uninstall complete message

17. **show_menu():**
    - print_logo
    - Show 3 options with bilingual labels:
      ```
      1) Установить / Install
      2) Обновить / Update
      3) Удалить / Uninstall
      ```
    - Read choice via prompt()
    - Case on 1/2/3, call do_install/do_update/do_uninstall

18. **main() entry:**
    - `set -e` at script top (but handle pipe errors explicitly)
    - detect_lang
    - detect_tty
    - If `--install` arg: skip menu, run do_install directly (for non-interactive curl|sh use)
    - If `--update` arg: skip menu, run do_update directly
    - If `--uninstall` arg: skip menu, run do_uninstall directly
    - Otherwise: show_menu

**POSIX compliance rules:**
- Use `[ ]` not `[[ ]]`
- Use `. file` not `source file`
- Use `$(command)` not backticks
- No arrays — use plain variables
- No `local` in functions (some BusyBox ash supports it, but not guaranteed — OK to use since Entware ash does support it)
- Actually, `local` IS supported in BusyBox ash — use it for function-local variables
- printf for colors, not echo -e

**Error handling:**
- `set -e` for general errors
- Explicit `|| die "message"` after critical commands (curl, tar, sha256sum)
- Trap cleanup: `trap cleanup EXIT` to remove TMP_DIR on any exit
