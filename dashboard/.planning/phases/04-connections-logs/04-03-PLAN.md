**LogStream.tsx**

Virtualized log list with auto-scroll behavior (like Chrome DevTools: scrolling up pauses auto-scroll, button to scroll back to bottom).

Props: `{ entries: LogEntry[] }`

Implementation:

1. **Virtualization setup**:
   ```tsx
   const parentRef = useRef<HTMLDivElement>(null)
   const virtualizer = useVirtualizer({
     count: entries.length,
     getScrollElement: () => parentRef.current,
     estimateSize: () => 36,  // compact card height estimate
     overscan: 15,
     useFlushSync: false,  // Avoid React 19 warnings
   })
   ```

   Use `measureElement` for dynamic row heights (cards with fields will be taller):
   ```tsx
   <div ref={virtualizer.measureElement} data-index={virtualRow.index}>
     <LogCard entry={entries[virtualRow.index]} />
   </div>
   ```

2. **Auto-scroll logic**:

   Track whether user is at the bottom of the scroll container:
   ```tsx
   const [isAtBottom, setIsAtBottom] = useState(true)
   const isAtBottomRef = useRef(true)  // ref for use in scroll handler without re-render
   ```

   On scroll event of `parentRef`:
   ```tsx
   const handleScroll = useCallback(() => {
     const el = parentRef.current
     if (!el) return
     const threshold = 50  // px from bottom
     const atBottom = el.scrollHeight - el.scrollTop - el.clientHeight < threshold
     isAtBottomRef.current = atBottom
     setIsAtBottom(atBottom)
   }, [])
   ```

   Auto-scroll to bottom when new entries arrive AND user is at bottom:
   ```tsx
   useEffect(() => {
     if (isAtBottomRef.current && entries.length > 0) {
       // Use requestAnimationFrame for smooth scroll after render
       requestAnimationFrame(() => {
         virtualizer.scrollToIndex(entries.length - 1, { align: 'end' })
       })
     }
   }, [entries.length, virtualizer])
   ```

   **Important**: Use `entries.length` as dependency (not entries array reference) to avoid excessive re-scrolling. The virtualizer.scrollToIndex is the correct way to scroll in a virtualized list (not parentRef.scrollTo).

3. **"Scroll to bottom" button**:
   When `!isAtBottom`, show a floating button at bottom-right:
   ```tsx
   {!isAtBottom && (
     <button
       onClick={() => {
         virtualizer.scrollToIndex(entries.length - 1, { align: 'end', behavior: 'smooth' })
       }}
       className="absolute bottom-4 right-4 z-20 flex items-center gap-1.5 px-3 py-1.5 rounded-full bg-primary text-primary-foreground text-xs shadow-lg hover:bg-primary/90 transition-colors"
     >
       <ArrowDown className="h-3 w-3" />
       Вниз
     </button>
   )}
   ```

4. **Empty state**: When `entries.length === 0`, show centered placeholder:
   ```
   "Логи пусты. Ожидание данных..."
   ```
   Style: `text-muted-foreground text-sm`

5. **Container**: relative positioned for the floating button:
   ```tsx
   <div className="relative h-full">
     <div ref={parentRef} className="h-full overflow-auto" onScroll={handleScroll}>
       <div style={{ height: `${virtualizer.getTotalSize()}px`, position: 'relative' }}>
         {virtualizer.getVirtualItems().map((virtualRow) => (
           <div
             key={entries[virtualRow.index].id}
             ref={virtualizer.measureElement}
             data-index={virtualRow.index}
             style={{
               position: 'absolute',
               top: 0,
               left: 0,
               width: '100%',
               transform: `translateY(${virtualRow.start}px)`,
             }}
           >
             <LogCard entry={entries[virtualRow.index]} />
           </div>
         ))}
       </div>
     </div>
     {/* Floating scroll-to-bottom button */}
   </div>
   ```
