Embed approach: root-level embed.go + internal/spa/spa.go accepts fs.FS parameter.

1. Create `embed.go` at dashboard root directory:
   ```go
   package xkeenui

   import "embed"

   //go:embed all:dist
   var DistFS embed.FS
   ```
   The root package name is `xkeenui` (derived from module path). This file is importable by cmd/antigravity/main.go as `import xkeenui "github.com/mewbing/XKeen-UI-Xmeow"`.

2. Create minimal `dist/index.html` placeholder so embed compiles:
   ```html
   <!DOCTYPE html><html><body>Run npm run build to generate SPA</body></html>
   ```

3. Create `internal/spa/spa.go`:
   - `func NewSPAHandler(distFS fs.FS) http.Handler` -- accepts an fs.FS parameter (the embedded dist sub-filesystem from root package)
   - Uses `fs.Sub(distFS, "dist")` to strip the dist/ prefix
   - Creates `http.FileServer(http.FS(sub))`
   - Pre-reads index.html content at init time for SPA fallback
   - On request: check if file exists via `fs.Stat(sub, path)` -- if yes, serve file; if no, serve index.html with Content-Type text/html and Cache-Control no-cache
   - For "/" path, serve index.html directly

4. Create `internal/server/middleware.go`:
   - `AuthMiddleware(getSecret func() string) func(http.Handler) http.Handler` -- chi-compatible middleware
   - Checks `Authorization: Bearer {token}` header OR `X-API-Key: {token}` header
   - Compares against secret from getSecret() (which reads mihomo config.yaml secret field)
   - If secret is empty (not configured), skip auth (allow all) -- per RESEARCH open question #4
   - If secret mismatch, return 401 `{"error": "Unauthorized"}`
   - Skip auth for OPTIONS requests (CORS preflight)

5. Create `internal/server/routes.go`:
   - `func NewRouter(cfg *config.AppConfig, spaHandler http.Handler) *chi.Mux`
   - Global middleware: chi middleware.Logger, middleware.Recoverer, middleware.RealIP
   - CORS middleware (dev mode only, per GOBK-07): AllowedOrigins ["*"], AllowedMethods ["GET","PUT","POST","DELETE","OPTIONS"], AllowedHeaders ["Accept","Authorization","Content-Type","X-API-Key"], AllowCredentials true
   - Auth middleware on /api/* group (except /api/health)
   - Route group `/api` with all endpoint placeholders (only /api/health implemented now; rest will be added in plans 02-04). Add comment placeholder stubs for the remaining 14 endpoints.
   - WebSocket route `/ws/logs` placeholder (plan 03)
   - Mihomo reverse proxy `/api/mihomo/*` placeholder (plan 04)
   - `r.NotFound(spaHandler.ServeHTTP)` as LAST -- SPA fallback

6. Create `internal/handler/health.go`:
   - `func Health(w http.ResponseWriter, r *http.Request)` -- returns `{"status": "ok"}` with 200

7. Create `internal/server/server.go`:
   - `type Server struct` with http.Server, config, router
   - `func New(cfg *config.AppConfig, distFS fs.FS) *Server` -- creates chi router, SPA handler, wires everything
   - `func (s *Server) Start() error` -- ListenAndServe
   - `func (s *Server) Shutdown(ctx context.Context) error` -- graceful shutdown

8. Create `cmd/antigravity/main.go`:
