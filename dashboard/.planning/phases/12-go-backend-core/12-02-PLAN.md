```go
type Handlers struct {
    cfg *config.AppConfig
}
func NewHandlers(cfg *config.AppConfig) *Handlers { return &Handlers{cfg: cfg} }
```

**internal/handler/config.go** -- port from server.py lines 247-292:
- `(h *Handlers) GetConfig(w, r)`:
  - Read file at `h.cfg.MihomoConfigPath`
  - If not found: 404 `{"error": "Config file not found"}`
  - If error: 500 `{"error": "..."}`
  - Success: 200 `{"content": "...file content..."}`
- `(h *Handlers) PutConfig(w, r)`:
  - Decode JSON body expecting `{"content": "..."}`. If missing: 400 `{"error": "Missing \"content\" field in request body"}`
  - Validate YAML: use `goccy/go-yaml` Unmarshal into `interface{}`. If invalid: 400 `{"error": "Invalid YAML: ..."}`
  - Backup: `backup.CreateBackup(h.cfg.MihomoConfigPath, "config", ".yaml", h.cfg.BackupDir)`. If error: 500
  - Write file with `os.WriteFile`. MkdirAll parent first. If error: 500
  - Success: 200 `{"message": "Config saved", "backup": "path/to/backup"}`

**internal/handler/xkeen.go** -- port from server.py lines 299-350:
- `(h *Handlers) GetXkeenFile(w, r)`:
  - Get `filename` from chi.URLParam(r, "filename")
  - Validate against `h.cfg.XkeenFiles` map. If not found: 404 `{"error": "Unknown file"}`
  - Build path: `filepath.Join(h.cfg.XkeenDir, actualFilename)`
  - Read file. If FileNotFoundError: return 200 `{"content": "", "filename": actualFilename}` (NOT an error -- Flask returns empty)
  - Success: 200 `{"content": "...", "filename": "ip_exclude.lst"}`
- `(h *Handlers) PutXkeenFile(w, r)`:
  - Get filename, validate against map. Decode JSON body.
  - Backup existing file: `backup.CreateBackup(filePath, filename, ".lst", h.cfg.BackupDir)`
  - Write new content. MkdirAll parent.
  - Success: 200 `{"message": "{filename} saved"}`

**internal/handler/service.go** -- port from server.py lines 106-216:
- `LOG_MAX_LINES = 200` constant
- `trimLogFile(path string, maxLines int)` -- keep only last maxLines in log file (identical to Flask `_trim_log_file`)
- `(h *Handlers) ServiceAction(w, r)`:
  - Get action from chi.URLParam(r, "action")
  - Validate: must be "start", "stop", or "restart". Else 400.
  - Build command: use `exec.Command(h.cfg.XkeenBin, "-"+action)` -- full path from config (NOT bare "xkeen" -- per RESEARCH pitfall #6, PATH on Entware may not include /opt/sbin)
  - Log path: `filepath.Join(h.cfg.XkeenLogDir, "error.log")`
  - MkdirAll log dir, trimLogFile before writing
  - Redirect stdout+stderr to log file (open in append mode), set proc.Stdout and proc.Stderr
  - Run with 60-second timeout context. If exit code != 0: 500. If timeout: 500.
  - Success: 200 `{"status": "ok"}`
- `(h *Handlers) ServiceStatus(w, r)`:
  - Try `exec.Command("pidof", "mihomo")` then `exec.Command("pidof", "xray")`
  - If either returns exit 0, parse first PID from stdout
  - Return 200 `{"running": true, "pid": 12345}` or `{"running": false, "pid": null}`
  - CRITICAL: JSON null for pid when not running. Use `*int` pointer type.

  - Return 200 `{"xkeen": "...", "dashboard": "..."}`

**internal/handler/system.go** -- port from server.py lines 356-437:
- CPU state: `type cpuState struct { prevIdle, prevTotal int64 }` -- package-level var with sync.Mutex
- `(h *Handlers) SystemCPU(w, r)`:
  - Read `/proc/stat` first line, parse values
  - Delta calculation against previous reading (identical to Flask `_prev_cpu` pattern)
  - If first reading (prev total == 0): return `{"cpu": 0}`
  - Calculate: `usage = (1 - deltaIdle/deltaTotal) * 100`, round to 1 decimal
  - Return 200 `{"cpu": 45.2}`. On error: `{"cpu": 0, "error": "..."}`
- `(h *Handlers) SystemNetwork(w, r)`:
  - External IP: try curl to services in order: ifconfig.me, icanhazip.com, api.ipify.org, ip.sb
  - Use `exec.CommandContext` with 3-second timeout per service, 5-second total timeout
  - Validate IP with regex `^[\d.]+$|^[0-9a-fA-F:]+$`
  - If IP found: get info from `http://ip-api.com/json/{ip}?fields=country,city,isp,query` via curl
  - Read `/proc/uptime` for system uptime (first field, convert to int seconds)
  - Return 200 `{"ip": "1.2.3.4", "info": {"country":"RU","city":"Moscow","isp":"ISP","query":"1.2.3.4"}, "uptime": 123456}`
  - Use `interface{}` for info field to preserve exact JSON structure from ip-api.com
  - Null values: ip can be null, info can be null, uptime can be null -- use pointer types
