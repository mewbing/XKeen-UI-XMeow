**internal/proxy/mihomo.go** -- based on RESEARCH pattern 4:
- `func NewMihomoProxy(cfg *config.AppConfig) http.Handler`:
  - Read mihomo address: `config.GetMihomoExternalController(cfg.MihomoConfigPath)` -- returns "host:port" (e.g., "127.0.0.1:9090")
  - Read mihomo secret: `config.GetMihomoSecret(cfg.MihomoConfigPath)` -- returns secret string or empty
  - Build target URL: `http://{address}`
  - Create `httputil.ReverseProxy` with `Rewrite` function (NOT Director -- per RESEARCH anti-pattern):
    ```go
    proxy := &httputil.ReverseProxy{
        Rewrite: func(pr *httputil.ProxyRequest) {
            pr.SetURL(target)
            // Strip /api/mihomo prefix from path
            pr.Out.URL.Path = strings.TrimPrefix(pr.Out.URL.Path, "/api/mihomo")
            if pr.Out.URL.Path == "" {
                pr.Out.URL.Path = "/"
            }
            // Inject Authorization header with mihomo secret
            if secret != "" {
                pr.Out.Header.Set("Authorization", "Bearer "+secret)
            }
        },
    }
    ```
  - Return proxy (Rewrite handles all path manipulation, no StripPrefix needed)
  - If mihomo address is empty or config not found: return a handler that responds 503 `{"error": "Mihomo not configured"}`
  - Log the proxy target on creation: `log.Printf("Mihomo reverse proxy -> %s", target)`
