<task type="auto">
  <name>Task 2: Release workflow с матрицей кросс-компиляции</name>
  <files>.github/workflows/release.yml</files>
  <action>
Создать `.github/workflows/release.yml` -- полный release pipeline по тегу:

**Триггер и permissions:**
- on: push tags ['v*.*.*']
- permissions: contents: write (нужно для создания Release)

**Job 1: build-frontend**
- runs-on: ubuntu-latest
- Steps:
  a) actions/checkout@v6
  b) pnpm/action-setup@v4
  c) actions/setup-node@v6 с node-version: '22', cache: 'pnpm'
  d) `pnpm install --frozen-lockfile`
  e) `pnpm run build`
  f) actions/upload-artifact@v4 с name: 'dist', path: 'dist/'

**Job 2: build-binaries** (needs: build-frontend)
- runs-on: ubuntu-latest
- strategy.matrix.include -- 5 архитектур:
  | goos | goarch | gomips | goarm | suffix |
  |------|--------|--------|-------|--------|
  | linux | arm64 | - | - | linux_arm64 |
  | linux | mipsle | softfloat | - | linux_mipsle |
  | linux | mips | softfloat | - | linux_mips |
  | linux | amd64 | - | - | linux_amd64 |
  | linux | arm | - | 7 | linux_armv7 |

- Steps:
  a) actions/checkout@v6
  b) actions/download-artifact@v4 с name: 'dist', path: 'dist/'
  c) actions/setup-go@v6 с go-version: '1.25'
  d) Extract version: `echo "version=${GITHUB_REF_NAME#v}" >> $GITHUB_OUTPUT`
  e) Build: `go build -ldflags "-s -w -X main.Version=$VERSION" -o xmeow-ui ./cmd/xmeow-ui/`
     env: CGO_ENABLED=0, GOOS, GOARCH, GOMIPS (если задан), GOARM (если задан)
     ВАЖНО: GOMIPS и GOARM задавать через `env:` а не через `${{ matrix.gomips || '' }}` --
     пустая строка в env не устанавливает переменную, Go использует дефолт
  f) UPX сжатие: crazy-max/ghaction-upx@v3 с version: latest, files: 'xmeow-ui', args: '--best --lzma'
  g) Package: `tar -czf xmeow-ui_$VERSION_$SUFFIX.tar.gz xmeow-ui README.md`
     БЕЗ вложенной папки -- файлы в корне архива (per user decision)
  h) actions/upload-artifact@v4 с name: 'binary-${{ matrix.suffix }}', path: '*.tar.gz'

**Job 3: release** (needs: [build-frontend, build-binaries])
- runs-on: ubuntu-latest
- Steps:
  a) actions/download-artifact@v4 с pattern: 'binary-*', merge-multiple: true
  b) actions/download-artifact@v4 с name: 'dist', path: 'dist/'
  c) Package dist: `tar -czf dist.tar.gz -C dist/ .`
  d) Checksums: `sha256sum *.tar.gz > checksums.txt`
  e) softprops/action-gh-release@v2:
     - name: "XMeow UI ${{ github.ref_name }}"
     - generate_release_notes: true
     - prerelease: ${{ contains(github.ref_name, '-') }}
     - files: *.tar.gz, checksums.txt

ВАЖНО -- анти-паттерны, которых надо избегать:
- НЕ собирать frontend в каждом matrix job (один раз в build-frontend, передать через artifact)
- НЕ включать checksums.txt в сам checksums.txt (sha256sum *.tar.gz, не sha256sum *)
- НЕ создавать вложенную папку в tar.gz (файлы в корне)
- ВСЕГДА задавать CGO_ENABLED=0 явно (Go включает CGO по умолчанию на amd64)
- ВСЕГДА задавать GOMIPS=softfloat для mips/mipsle (дефолт hardfloat = crash на роутере)
