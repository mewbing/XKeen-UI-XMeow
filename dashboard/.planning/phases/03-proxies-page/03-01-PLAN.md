    1. Создать `src/stores/proxies.ts` — VOLATILE store (без persist):
       ```typescript
       import { create } from 'zustand'
       import type { Proxy } from '@/lib/mihomo-api'
       import { fetchProxies, selectProxy, fetchProxyDelay, fetchGroupDelay } from '@/lib/mihomo-api'
       import { toast } from 'sonner'
       ```

       **Интерфейсы:**
       ```typescript
       interface DelayResult {
         delay: number      // ms, 0 = timeout
         testedAt: number   // Date.now()
       }

       interface ProxiesState {
         // Данные
         proxyMap: Record<string, Proxy>
         groupNames: string[]
         delayCache: Record<string, DelayResult>

         // UI состояние
         expandedGroups: Set<string>
         loading: boolean
         testingGroups: Set<string>
         testingProxies: Set<string>

         // Actions
         fetchAllProxies: () => Promise<void>
         toggleExpand: (groupName: string) => void
         selectProxyInGroup: (groupName: string, proxyName: string) => Promise<void>
         testProxyDelay: (proxyName: string) => Promise<number>
         testGroupDelay: (groupName: string) => Promise<void>
         testAllGroups: () => Promise<void>
         isDelayCacheValid: (proxyName: string) => boolean
       }
       ```

       **Реализация actions:**

       a) `fetchAllProxies`:
          - set loading: true
          - Вызвать `fetchProxies()`
          - Извлечь группы: объекты с `all !== undefined`, исключить `name === 'GLOBAL'` и `hidden === true`
          - Отсортировать groupNames по алфавиту
          - set proxyMap (весь Record), groupNames, loading: false
          - При ошибке: toast.error с сообщением, set loading: false

       b) `toggleExpand(groupName)`:
          - Клонировать Set expandedGroups
          - Если есть — удалить (свернуть), если нет — добавить (развернуть)
          - set expandedGroups: новый Set

       c) `selectProxyInGroup(groupName, proxyName)`:
          - Оптимистичное обновление: сразу set proxyMap[groupName].now = proxyName
          - Вызвать `selectProxy(groupName, proxyName)`
          - При успехе: toast.success(`Переключено на ${proxyName}`)
          - При ошибке: откатить proxyMap[groupName].now к предыдущему значению, toast.error

       d) `testProxyDelay(proxyName)`:
          - Проверить кеш: если запись есть и (Date.now() - testedAt) < 15000 — вернуть cached delay
          - Добавить proxyName в testingProxies Set
          - Вызвать `fetchProxyDelay(proxyName)`
          - Обновить delayCache[proxyName] = { delay, testedAt: Date.now() }
          - Удалить proxyName из testingProxies Set
          - Вернуть delay
          - При ошибке: delayCache[proxyName] = { delay: 0, testedAt: Date.now() }, удалить из testingProxies

       e) `testGroupDelay(groupName)`:
          - Добавить groupName в testingGroups Set
          - Вызвать `fetchGroupDelay(groupName)`
          - Результат — Record<string, number>: для каждого ключа обновить delayCache
          - Удалить groupName из testingGroups Set
          - При ошибке: toast.error, удалить из testingGroups

       f) `testAllGroups()`:
          - Последовательно (for..of) вызвать testGroupDelay для каждого groupName из groupNames
          - НЕ параллельно — чтобы не перегрузить роутер
          - toast.success по завершении всех тестов

       g) `isDelayCacheValid(proxyName)`:
          - Проверить delayCache[proxyName] существует и Date.now() - testedAt < 15000

       **ВАЖНО:** Set<string> для expandedGroups, testingGroups, testingProxies. При создании store инициализировать как `new Set()`.
