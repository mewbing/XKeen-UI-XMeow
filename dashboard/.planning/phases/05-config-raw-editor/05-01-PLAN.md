**A. Создать `src/stores/config-editor.ts`** -- Zustand store (volatile, без persist) по паттерну существующих stores (logs.ts, connections.ts):

```typescript
type TabId = 'config' | 'ip_exclude' | 'port_exclude' | 'port_proxying'

interface TabState {
  original: string        // Содержимое с сервера (для dirty-сравнения и diff)
  current: string         // Текущее содержимое в редакторе
  dirty: boolean          // original !== current
  language: 'yaml' | 'plaintext'
  validation: ValidationResult
  loading: boolean
}

interface ValidationResult {
  valid: boolean
  error?: { message: string; line: number; column: number }
}

interface LogEntry {
  id: number
  time: string
  level: string
  message: string
}
```

Store должен содержать:
- `activeTab: TabId` -- текущий активный таб (по умолчанию 'config')
- `tabs: Record<TabId, TabState>` -- состояние каждого из 4 табов. Инициализировать с пустыми строками, dirty: false, loading: false, validation: { valid: true }. Язык: 'config' -> 'yaml', остальные 3 -> 'plaintext'.
- `logStreaming: boolean` -- идёт ли стриминг логов (по умолчанию false)
- `logEntries: LogEntry[]` -- записи лога (буфер для лог-панели)
- `logNextId: number` -- счётчик ID для записей

Actions:
- `setActiveTab(tab: TabId)` -- переключить активный таб
- `setContent(tab: TabId, content: string)` -- обновить current содержимое + пересчитать dirty (current !== original)
- `setOriginal(tab: TabId, content: string)` -- установить original и current одновременно (при загрузке с сервера), dirty = false
- `setLoading(tab: TabId, loading: boolean)` -- флаг загрузки
- `markSaved(tab: TabId)` -- после успешного сохранения: original = current, dirty = false
- `setValidation(tab: TabId, result: ValidationResult)` -- обновить результат валидации
- `startLogStream()` -- logStreaming = true, logEntries = [] (очистить при новом Apply)
- `stopLogStream()` -- logStreaming = false
- `addLogEntry(entry: Omit<LogEntry, 'id'>)` -- добавить запись, id = logNextId++. Ring buffer: max 500 записей.
- `clearLogs()` -- logEntries = [], logNextId = 0
- `hasDirtyTabs()` -- computed: проверить есть ли хотя бы один dirty таб (для navigation guard)
