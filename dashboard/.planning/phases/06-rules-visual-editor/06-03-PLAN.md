- Fields:

  - **Value** — Input text field (disabled for MATCH type)
  - **Target proxy-group** — Select dropdown populated from `proxyGroups` prop, default = `defaultTarget`
  - **no-resolve** — Checkbox/Switch (only shown for IP-based rule types: IP-CIDR, IP-CIDR6, SRC-IP-CIDR, GEOIP)

- Validation: type required, value required (except MATCH), target required
- Reset form on close

  - **Target proxy-group** — Select from proxyGroups

- Uses shadcn/ui AlertDialog
- Props: `open: boolean`, `onOpenChange: (v: boolean) => void`, `title: string`, `description: string`, `onConfirm: () => void`, `onCancel: () => void`
- Shows warning icon (AlertTriangle from lucide-react) in yellow/amber
- Title and description dynamically set based on the danger type
- Two buttons: "Отменить" (cancel — reverts the move) and "Продолжить" (confirm — applies the move)
